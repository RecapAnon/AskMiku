🗄️ Database Management

Manage your Ask Miku knowledge base

📥 Import Database

Load database from a previously exported JSON file. This will populate the vector database with pre-calculated embeddings for instant startup.

📤 Export Database

Export the current database state to a JSON file. This includes all vector embeddings and metadata. Use this file for backup or to speed up future initializations.

🗑️ Clear Database

Remove all data from the vector database. This action cannot be undone. You'll need to import or load dummy data again after clearing.
